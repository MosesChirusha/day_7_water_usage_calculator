## Top Water Saving Tips

1. **Reduce shower time** to 5-7 minutes → Save 30L/shower
2. **Fix leaking taps** → Save up to 20L/day
3. **Turn off tap** while soaping/brushing → Save 6L/minute
4. **Use bucket for car washing** → Save 200L/wash
5. **Full loads only** for laundry → Save 50L/load
6. **Water garden early morning** → Less evaporation
7. **Dual-flush toilet** or bottle in tank → Save 3L/flush
8. **Collect greywater** for plants/cleaning
9. **Basin for dishes** not running water → Save 50%
10. **Replace bath with shower** → Save 30L
